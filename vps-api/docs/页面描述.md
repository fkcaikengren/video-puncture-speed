- **页面路由**：`/login` `

##### 3. 速度分析页
- **页面路由**：`/video/analysis`
- **页面结构与功能描述**：
	- **query参数**：`id`表示视频的id. 
    - **左侧-视频展示区**：
        - 拆分成两个视频展示，上面是原视频区，下面是“标记视频”
        - 原视频: 上传的原始视频。
        - “标记视频”:（带分割 Mask 和速度悬浮字体的合成视频）。
    - **右侧-数据看板区**：
        - **动态曲线图 (ECharts)**：
	        - 位置：右侧-上方
            - 横轴时间，纵轴速度。
            - 鼠标 Hover 互动：十字准星显示当前点位精确速度。
            - ~~**联动功能**：点击曲线上的点，视频播放器自动跳转到对应的帧。~~
        - **核心指标卡片**：
	        - 位置：右侧-下方
	        - 展示分析出的 `[start, end]` 区间、初始速度、平均速度。
	        - 「比较按钮」：点击跳转到
    

##### 4. 视频库列表页
- **页面路由**：`/video/list`
- **页面结构与功能描述**：
    - **搜索栏**：按名称、分类、上传时间、处理状态（待处理/处理中/已完成/失败）筛选。
    - **数据卡片/列表**：
        - 展示视频缩略图、上传人和上传时间。
        - 状态标签：如果是“已完成”，显示缩略图右上角（绿三角）白色勾选号；如果是“进行中”则换色三角，省略号；“失败”则显示红色三角，感叹号。
        - 视频时长：右下角白色显示视频时长。
        - 每行最多5个列表（flex布局，响应式）
    - **点击视频**：跳转到
    - **hover视频**：视频左上角显示删除按钮
